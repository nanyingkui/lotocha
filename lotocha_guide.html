알리페이 (支付宝)
여권 인증 + 한국 카드 등록 — 가장 범용적
⌄
1
한국에서 미리 App Store / Google Play에서 Alipay 설치
2
가입 시 국가코드 +82 선택 → 한국 휴대폰 번호로 SMS 인증
3
앱 내 검색창에 "Tour Pass" 입력 → 여권 사진 업로드 → 여권번호 입력 (1~3분 인증)
4
Tour Pass → Visa/Mastercard 한국 카드 등록 (충전 없이 카드 직접 결제)
5
중국 도착 후 QR 스캔 또는 코드 보여주기로 결제
⚠️ 인증에 최대 1~3일 소요. 출발 3일 전에 미리 완료 필수.
200위안 이상 결제 시 약 3% 수수료 발생 가능.
알리페이 다운로드 / 공식 사이트 →
💚
위챗페이 (WeChat Pay)
위챗 앱 내 결제 기능 — 카드 등록 필요
⌄
1
App Store / Google Play에서 WeChat(微信) 설치 → 한국 번호로 가입
2
Me → Services → Wallet → Cards → Add Card
3
해외 카드 등록: VISA/Mastercard/JCB/AMEX → 카드번호·만료일·CVV·여권이름 입력
4
실명인증 (여권 정보 입력 + 얼굴 인식 요청될 수 있음)
5
결제: Scan QR 또는 My QR Code 제시
⚠️ 외국인은 개인 간 송금 기능 제한. 결제 전용으로만 사용 가능.

디디택시 (DiDi) — 알리페이 내
알리페이 앱에서 바로 호출 — 중국어 불필요
⌄
1
알리페이 앱 열기 → 홈화면 상단 검색창에 "滴滴出行" 또는 "Didi" 입력
2
목적지 입력: 영문 또는 중문 모두 지원. 고덕지도에서 목적지 이름 복사해서 붙여넣기
3
차량 종류 선택 후 Confirm → 드라이버 배차 확인
4
탑승 후 결제는 알리페이 자동 결제 (내리면 자동 차감)
